Polymorphism
~~~~~~

/*
 * The word polymorphism means having many forms.
 * 
 * The word “poly” means many and “morphs” means forms, 
 * So it means many forms.
 * 
 * Polymorphism allows to perform a single action in different ways.
 * Polymorphism allows to define one interface and have multiple implementations.
 * 
 * Polymorphism in Java is mainly divided into two types:
 * 1. Compile-time Polymorphism
 * 2. Runtime Polymorphism
 */
